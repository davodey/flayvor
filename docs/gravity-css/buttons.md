## Examples
Gravity includes several predefined button styles, each serving its own semantic purpose, with a few extras thrown in for more control.
A button triggers a single action in a task flow. We use three types of button styles: Default, Primary, and Attention.
<div class="box">
<button class="gr-btn" type="button">Button</button>
<button class="gr-btn primary" type="button">Button</button>
<button class="gr-btn attention" type="button">Button</button>
</div>

```html
<button class="gr-btn" type="button">Button</button>
<button class="gr-btn primary" type="button">Button</button>
<button class="gr-btn attention" type="button">Button</button>
```
